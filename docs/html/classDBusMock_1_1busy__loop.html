Detailed Description

The busy_loop class is a basic polling loop that handles queued dbus operations.

it calls sd_bus_process and sd_bus_wait in a loop.

void DBusGlue::busy_loop::error_callback ( std::function< bool(int code, std::string const &message)> const &  cb)

error_callback Set an error callback that is called when an error occurs in the event loop.

The return of this function determines wheter the loop shall continue or not. true = continue, false = stop! Be warned, that if you dont use this function and set a callback, the loop will raise an exception within a thread and call std::terminate as a consequence. This is intentional, because silent failings are worse than total teardowns.

cbThe callback function.
